# Contributing to ehrtslib

Thank you for contributing.

- **Using the library in an app?** Start at [README.md](README.md) and
  [docs/README.md](docs/README.md).
- **Changing ehrtslib itself?** See [docs/maintainers/README.md](docs/maintainers/README.md)
  (codegen, tests, agents, roadmap).

Do not put maintainer-only notes in user-facing guides under `docs/` (except
`docs/maintainers/`).
