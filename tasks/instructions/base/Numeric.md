# Instruction: Implementing the `Numeric` Class

## 1. Description

The `Numeric` class is an abstract ancestor for numeric types with arithmetic operations.

-   **Reference:** [openEHR BASE - Numeric](https://specifications.openehr.org/releases/BASE/latest/foundation_types.html#_numeric_class)

## 2. Behavior

### 2.1. Abstract Methods

-   **`add(other: Numeric): Numeric`** - Addition
-   **`subtract(other: Numeric): Numeric`** - Subtraction
-   **`multiply(other: Numeric): Numeric`** - Multiplication
-   **`divide(other: Numeric): Numeric`** - Division
-   **`negative(): Numeric`** - Negation
