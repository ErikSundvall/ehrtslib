# Instruction: Implementing the `Ordered_Numeric` Class

## 1. Description

The `Ordered_Numeric` class combines Ordered and Numeric - numbers with both comparison and arithmetic.

-   **Reference:** [openEHR BASE - Ordered_Numeric](https://specifications.openehr.org/releases/BASE/latest/foundation_types.html#_ordered_numeric_class)

## 2. Behavior

Inherits all operations from both Ordered and Numeric.
