# Instruction: Implementing the `Temporal` Class

## 1. Description

The `Temporal` class is an abstract parent for date/time types.

-   **Reference:** [openEHR BASE - Temporal](https://specifications.openehr.org/releases/BASE/latest/foundation_types.html#_temporal_class)

## 2. Behavior

Abstract parent for Iso8601_date, Iso8601_time, Iso8601_date_time.

### 2.1. Common Operations

-   **`is_partial(): Boolean`** - Check if partially specified
-   **`is_extended(): Boolean`** - Check if extended format
