# Instruction: Implementing the `VERSION_STATUS` Class

## 1. Description

The `VERSION_STATUS` class is an enumeration of version lifecycle states.

- **Values:** trunk, branch
- **Reference:**
  [openEHR BASE - VERSION_STATUS](https://specifications.openehr.org/releases/BASE/latest/base_types.html#_version_status_class)

## 2. Behavior

Indicates version position in version tree.
