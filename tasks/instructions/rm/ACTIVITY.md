# Instruction: Implementing the `ACTIVITY` Class

## 1. Description

ACTIVITY defines a single activity within an INSTRUCTION.

- **Reference:**
  [openEHR RM - ACTIVITY](https://specifications.openehr.org/releases/RM/latest/ehr.html#_activity_class)

## 2. Behavior

- `description: ITEM_STRUCTURE` - Activity description
- `timing: DV_PARSABLE` - Timing specification
- `action_archetype_id: String` - Corresponding ACTION archetype
