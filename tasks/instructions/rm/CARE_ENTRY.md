# Instruction: Implementing the `CARE_ENTRY` Class

## 1. Description

CARE_ENTRY extends ENTRY for clinical care entries.

- **Reference:**
  [openEHR RM - CARE_ENTRY](https://specifications.openehr.org/releases/RM/latest/ehr.html#_care_entry_class)

## 2. Behavior

- `protocol: ITEM_STRUCTURE` - How observation/action was performed
- `guideline_id: OBJECT_REF` - Reference to guideline

### 2.2. Subclasses

- OBSERVATION
- EVALUATION
- INSTRUCTION
- ACTION
