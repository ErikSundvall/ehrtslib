# Instruction: Implementing the `INSTRUCTION_DETAILS` Class

## 1. Description

INSTRUCTION_DETAILS links an ACTION to its originating INSTRUCTION.

-   **Reference:** [openEHR RM - INSTRUCTION_DETAILS](https://specifications.openehr.org/releases/RM/latest/ehr.html#_instruction_details_class)

## 2. Behavior

- `instruction_id: LOCATABLE_REF` - Reference to INSTRUCTION
- `activity_id: String` - Activity archetype node ID
- `wf_details: ITEM_STRUCTURE` - Workflow details
