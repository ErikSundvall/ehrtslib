# Instruction: Implementing the `ITEM_STRUCTURE` Class

## 1. Description

ITEM_STRUCTURE is abstract parent for all item structures.

- **Reference:**
  [openEHR RM - ITEM_STRUCTURE](https://specifications.openehr.org/releases/RM/latest/data_structures.html#_item_structure_class)

## 2. Behavior

Abstract class, no additional properties.

### 2.2. Subclasses

- ITEM_TREE (hierarchical)
- ITEM_LIST (linear)
- ITEM_TABLE (tabular)
- ITEM_SINGLE (single element)
