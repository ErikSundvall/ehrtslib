# Phase 3 Task List: BASE Package

This task list outlines the steps for Phase 3 of the project, focusing on the `BASE` package. The goal is to gather information, create implementation instructions, and write failing tests for the core classes.

## 1. BASE Package Implementation and Testing

For each of the following classes in the `BASE` package:
- Create a markdown instruction file in `tasks/instructions/base/` named after the class (e.g., `Integer.md`).
- Add failing tests for the class to `tests/base.unimplemented.test.ts`.
